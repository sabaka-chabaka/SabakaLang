### SabakaLang is a simple .NET runtime lang

## Prerequesities

.NET 9.0

Visual Studio Code

## How to install

- download installer and .vsix extension
- install sabakalang from installer
- start VS code
- install extension from VSIX
- go to extentsions settings
- set paths to LSP, Compiler, EXE Builder
- now you can start make you first sabakalang programs!
